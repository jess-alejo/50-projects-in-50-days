Frequently Asked Questions

Is a computer science degree necessary?

If you are a student entering college or passionate about programming, then taking a CS degree would be a great help.

I'm not a programmer, how do I start?

It depends on your interest. If you want to create websites, you may start with basic HTML, CSS, and JavaScript. If you want to be a game developer, then you may consider learning Java or C++. Always start with the basics.
